## 新版本内容

- 好友联系人列表支持分批加载，进入页面后先显示部分联系人，继续向下滚动时自动加载更多。
- 优化好友界面显示效果，联系人行宽保持一致，列表视觉更稳定。
- 联系人列表排序调整为更接近抖音会话逻辑：按最后消息时间排序，在线状态和最近活跃时间不再导致列表频繁跳动。
- 好友列表在未点开聊天记录时，会使用服务端最近互动时间作为排序兜底。
- 私信接收连接增加自动重连能力，连接断开后会自动尝试恢复。
- Rust 版私信接收状态显示更准确，修复 WebSocket 实际已连接但界面仍显示“接收未连接”的问题。
- 支持更多私信消息类型解析，包括收藏表情、贴纸图片、评论分享卡片和 POI / 生活服务视频分享卡片。
- 修复登录态检测误判问题，个人资料接口临时不可用时不会直接判定 Cookie 失效。
